# L4 Language Documentation

Welcome to the L4 language documentation. L4 is a domain-specific language for law that enables computer-readable formalizations of contracts, legislation, and regulations.

## Getting Started

New to L4? Start with:

1. **[Download the VSCode extension](https://marketplace.visualstudio.com/items?itemName=Legalese.l4-vscode)** or [code online](https://jl4.legalese.com)
2. **[Foundation Course](courses/foundation/README.md))** - Get oriented
3. **[Quickstart](courses/foundation/quickstart.md)** - Your first rules in L4
4. **[Tutorial: Your First L4 File](tutorials/getting-started/first-l4-file.md)** - Hands-on practice
