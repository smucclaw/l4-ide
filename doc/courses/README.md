## Learning Paths

### For Legal Professionals
1. Foundation Course (complete all modules)
2. Advanced: Module A2 (AI Ingestion)
3. Advanced: Module A5 (Rebuttable Presumptions)
4. Advanced: Module A11 (Regulative Rules)

### For Developers
1. Foundation: Modules 0-4
2. Advanced: Module A1 (VSCode Workflow)
3. Advanced: Module A4 (Decision Services)
4. Advanced: Module A6-A8 (Integration and Architecture)

### For Legal Engineers
1. Complete Foundation Course
2. Complete Advanced Course
3. Explore [Concepts](../concepts/README.md) for deeper understanding

## How to Use These Courses

### Self-Paced Learning
- Work through modules in order
- Complete exercises in each module
- Test your knowledge with the examples

### Classroom/Workshop Use
- Modules designed for 45-60 minute sessions
- Include discussion questions and group activities
- Instructor notes available in each module

### Reference While Coding
- Each module stands alone as reference material
- Use module READMEs to find specific topics quickly
