## How to Use These Courses

### Self-Paced Learning

- Work through modules in order
- Complete exercises in each module
- Test your knowledge with the examples

### Classroom/Workshop Use

- Modules designed for 45-60 minute sessions
- Include discussion questions and group activities
- Instructor notes available in each module

### Reference While Coding

- Each module stands alone as reference material
- Use module READMEs to find specific topics quickly
