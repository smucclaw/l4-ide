# Jurisdiction Library

Jurisdiction definitions for multi-jurisdictional legal rules. Uses ISO 3166 country codes. Can be imported into L4 files with `IMPORT jurisdiction`.

### Location

[jl4-core/libraries/jurisdiction.l4](https://github.com/smucclaw/l4-ide/blob/main/jl4-core/libraries/jurisdiction.l4)

### Features

- ISO 3166-1 alpha-2 country codes
- US state codes
- UK constituent country codes
- Canadian province codes
- Regional groupings (EU, etc.)
