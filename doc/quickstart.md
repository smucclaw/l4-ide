# Quickstart

This document is under construction.

## Requirements

- Haskell toolchain
  - [GHCup](https://www.haskell.org/ghcup/) - Haskell installer
  - GHC 9.8.4 (install via `ghcup tui`)
  - Cabal 3.10 or newer

After installing GHCup, run `ghcup tui` and set GHC to version `9.8.4`; press `i` to install and then `s` to set it as the default.
